Utilizado no aplicativo:
Coroutines, Room Database, clean code, arquitetura mvvm(model-view-view-model).

usar o aplicativo:
1. No botão cadastrar se pode cadastrar um produto com suas especificações.
2. Botão excluir se pode excluir um produto.
3. Ao cadastrar um produto vai direcionar a outra tela carregando uma lista.
Ao cadastrar o produto o mesmo ira aparecer no topo da lista, e os demais 
produtos vai aparecer conforme a pesquisa.
4. na lista ha um botão para se atualizar o produto e suas especificações.
